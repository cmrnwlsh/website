### Installation:

Install the rust toolchain @ https://www.rust-lang.org/tools/install

use "cargo install cargo-leptos" to install the cargo-leptos build tool

use "cargo leptos watch" for hot reloading dev webpage at localhost:3000

use "cargo leptos build --release" to build the web server and frontend with optimized size/speed

after building for release, run ./target/release/website from the project root folder
